# STM32 Embedded Systems Portfolio

This repository documents my journey in mastering embedded systems using STM32F446RE.

## Hardware Platform
- STM32 Nucleo F446RE

## Development Style
- Bare-metal register level programming
- No HAL abstraction (except optional comparison)
- FreeRTOS integration
- CAN communication
- Driver architecture separation

## Skill Areas Covered
- GPIO
- UART
- SPI
- I2C
- ADC
- Timers & PWM
- CAN Protocol
- FreeRTOS
- State Machines
- Distributed Systems

## Major Projects
- CAN ECU Simulation
- RTOS Based Irrigation Controller

## Debugging Tools Used
- ST-Link
- Logic Analyzer
- UART Logging

## Goal
To develop strong low-level embedded systems engineering skills for core placements.
